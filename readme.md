## algorithm-go
数据结构与算法的go语言版本，供大家阅读参考。

## 使用说明

- 请确保你已经安装了go语言开发环境
- 克隆项目

```bash
git clone https://github.com/go-zend/algorithm-go.git
```

- 运行相应的单元测试(示例)

```bash
cd tree && go test -v -run=TestToTree
```
